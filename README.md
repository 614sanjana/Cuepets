# Cuepets

Cuepets is a platform designed to streamline the management and maintenance of pet medical records. It targets pet owners, foster parents, animal lovers, and veterinarians (veterinarian view planned for future versions). The platform allows users to save appointments, upload and manage medical/vaccination records of their pets, add pets, access veterinary blogs, and adopt or rehome pets — addressing the increasing number of abandoned and stray animals.

## Features

- User login/signup (basic authentication)
- Add, edit, and view pet health records and vaccination schedules
- Schedule and save appointments for pets
- Add and manage multiple pets
- Post and manage veterinary blogs and articles
- Adopt and rehome pets through the platform
- Dashboard overview for pet owners

## Technology Stack

- **Backend:** Java, Spring Boot, MongoDB
- **Frontend:** React, Tailwind CSS

## Project Structure

- `Cuepets/React/cuepetsfe` — Frontend React application
- `Cuepets/src` — Backend Spring Boot application

## Getting Started

### Prerequisites

- Java JDK 17 or above installed and configured in your system PATH
- Maven installed and configured in your system PATH
- Node.js and npm installed

### Running Locally

1. Clone the repository:

   ```bash
   git clone <your-repository-url>
   cd Cuepets/React/cuepetsfe
````

2. Install frontend dependencies:

   ```bash
   npm install
   ```

3. Start frontend and backend concurrently:

   ```bash
   npm run dev
   ```

   This will start both React frontend and Spring Boot backend together.

## Environment Variables / Configurations

No special environment variables are currently required for local development.
